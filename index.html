RaminTodo

RaminTodo is a comprehensive To-Do list application developed using Laravel, Vue.js, and Inertia.js. The application allows users to add new tasks, mark tasks as complete, edit or delete tasks with a smooth and user-friendly interface. It's ideal for personal task management and project tracking. This project demonstrates an understanding of back-end (Laravel), front-end (Vue.js), and seamless page navigation with Inertia.js. Users can also view completed tasks, and it features a responsive design for various device types. The project is hosted on GitHub for further inspection and collaboration.
